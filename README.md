# Marli Lucas — Portfolio

Product Designer & Manager portfolio site.

## Deploy to Vercel (free)

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) and sign in with GitHub
3. Click **Add New → Project**
4. Import your repository
5. Vercel will auto-detect the static site — click **Deploy**
6. Your portfolio will be live at `your-project.vercel.app`

### Or deploy from CLI

```bash
npm i -g vercel
cd portfolio
vercel
```

## Custom domain (optional)

In your Vercel project dashboard → Settings → Domains → add your custom domain.
